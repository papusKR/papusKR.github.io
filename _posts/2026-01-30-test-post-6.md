# 악성코드 분석과 위협 인텔리전스

악성코드 분석은 사이버 보안의 핵심 기술입니다.

## 분석 유형

### 1. 정적 분석 (Static Analysis)
```bash
# 파일 정보 확인
file suspicious.exe

# 문자열 추출
strings suspicious.exe | grep -i "http"

# 해시 값 계산
md5sum suspicious.exe
sha256sum suspicious.exe
```

### 2. 동적 분석 (Dynamic Analysis)
```bash
# 프로세스 모니터링
ps aux | grep suspicious

# 네트워크 연결 확인
netstat -ano | grep ESTABLISHED

# 시스템 콜 추적
strace -p <PID>
```

### 3. 샌드박스 분석
- Cuckoo Sandbox
- ANY.RUN
- Joe Sandbox

## 악성코드 행위 분석

### 파일 시스템 변경
```python
import os
import hashlib

def monitor_file_changes(directory):
    baseline = {}
    for root, dirs, files in os.walk(directory):
        for file in files:
            filepath = os.path.join(root, file)
            with open(filepath, 'rb') as f:
                file_hash = hashlib.sha256(f.read()).hexdigest()
            baseline[filepath] = file_hash
    return baseline
```

### 레지스트리 변경 (Windows)
```powershell
# 자동 실행 레지스트리 키 확인
Get-ItemProperty -Path "HKLM:\Software\Microsoft\Windows\CurrentVersion\Run"
Get-ItemProperty -Path "HKCU:\Software\Microsoft\Windows\CurrentVersion\Run"
```

## 위협 인텔리전스

### IOC (Indicators of Compromise)
- IP 주소
- 도메인
- 파일 해시
- URL 패턴

### MITRE ATT&CK 프레임워크
```yaml
Technique: T1059.001 - PowerShell
Tactic: Execution
Description: 공격자가 PowerShell을 사용하여 명령 실행
```

## 분석 도구

### 1. IDA Pro / Ghidra
역어셈블리 및 디컴파일 도구

### 2. Wireshark
네트워크 트래픽 분석

### 3. Volatility
메모리 포렌식 분석

```bash
# 메모리 덤프 분석
volatility -f memory.dmp imageinfo
volatility -f memory.dmp --profile=Win10x64 pslist
volatility -f memory.dmp --profile=Win10x64 netscan
```
